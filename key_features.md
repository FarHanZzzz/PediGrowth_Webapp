# Key Features: Deep Biomechanics

## 1) 33 Landmarks
Our edge AI tracks 33 critical skeletal landmarks frame-by-frame with high precision, including on low-end smartphone cameras.

## 2) 34 Metrics
The system automatically computes 34 distinct biomechanical metrics, including:
- Step length
- Cadence
- Joint angles
- Asymmetry scores

## 3) Quality Gating
If lighting is poor or camera angle is unsuitable, the AI rejects the video and guides the parent to recapture it.

No garbage in, garbage out.

## Additional Features (Beyond the Slide)

## 4) Clinician Packet PDF Export
The platform generates a structured clinician packet as a deterministic PDF for handoff and review.

## 5) Secure Share Links for Handoff
Clinician packets can be shared via secure links with expiration and optional access limits.

## 6) Smart Routing (Route A / Route B)
The system routes users based on ambulatory status and age to keep analysis clinically appropriate.

## 7) Non-Diagnostic AI Navigator
The AI assistant is policy-bounded, evidence-grounded, and automatically falls back to deterministic safe guidance in low-confidence conditions.

## 8) Connectivity-Aware Design
Core capture, quality checks, and pose extraction can run locally, while sync and sharing use online services when available.
